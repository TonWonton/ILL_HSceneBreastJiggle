# ILL_HSceneAnimationSpeed

## Description
HScene animation speed plugin / MOD for `Aicomi` / `アイコミ` and `Samabake Scramble` / `サマバケ！すくらんぶる`. Set custom animation speed during HScenes.

## Prerequisites
- [BepInEx Unity Il2Cpp](https://github.com/BepInEx/BepInEx)
- [BepisPlugins](https://github.com/IllusionMods/BepisPlugins)
- [BepInEx.ConfigurationManager Il2Cpp](https://github.com/BepInEx/BepInEx.ConfigurationManager)

#### OR
- [AC-HF_Patch](https://github.com/ManlyMarco/AC-HF_Patch) for AC
- [SVS-HF_Patch](https://github.com/ManlyMarco/SVS-HF_Patch) for SVS

## Installation
1. Install prerequisites for the correct game
2. Download the correct version from releases
3. Extract into game folder (.DLL file should be in `/GAME_FOLDER/BepInEx/plugins/`)

## Features
- Set custom animation speeds for the different states/animations
  - AC
    - WLoop min/max speed
	- SLoop min/max speed
	- OLoop min/max speed
  - SVS
    - Animation loop speed

## Note/known issues
- The orgasm animation loop is still the default speed
